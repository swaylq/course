空格的处理

有研究顯示，打字的時候不喜歡在中文和英文之間加空格的人，感情路都走得很辛苦，有七成的比例會在 34 歲的時候跟自己不愛的人結婚，而其餘三成的人最後只能把遺產留給自己的貓。畢竟愛情跟書寫都需要適時地留白。

為什麼你們就是不能加個空格呢？

常用字符的简单分类

  • 中文字符
  • 英文字符（半角）
  • 阿拉伯数字
  • 全角标点
  • 半角标点
1. 中文字符、英文字符、阿拉伯数字，任意两种字符之间添加空格（特殊名词除外，例如 ES2015）。

错误：

Sway的生日是9月13号，大家记得要送他生日礼物哦。

正确：

Sway 的生日是 9 月 13 号，大家记得要送他生日礼物哦。

错误：

我的电脑硬盘有100TB 大小。

正确：

我的电脑硬盘有 100 TB 大小。

2. 全角标点前后都不需要加空格。

错误：

她说 ：『玲珑骰子安红豆 』

正确：

她说：『玲珑骰子安红豆』

错误：

我想了想， 说：『我知。 』

正确：

我想了想，说：『我知。』

3. 半角标点，前标点（前括号，前引号）在前面加一个空格，其余标点，在后面加一个空格（句末除外），标点之间不需要加空格。

错误：

Eric( Cheng Chu)is a muscle man.He likes to show his muscle in public.

正确：

Eric (Cheng Chu) is a muscle man. He likes to show his muscle in public.

错误：

他(Gay? )到底是不是直男， 谁也不知道。

正确：

他（Gay?）到底是不是直男，谁也不知道。

正确：

他 (Gay?) 到底是不是直男，谁也不知道。

4. 超链接两边加空格，遇到全角字符可不加

错误：

分享一个同性交友网站，你会爱上它的。

正确：

分享一个 同性交友网站，你会爱上它的。

总结

1. 中文、英文字符、数字，任意两种之间添加空格（特殊名词除外，例如 ES2015）。

2. 全角标点前后都不需要加空格。

3. 半角标点，前标点（前括号，前引号）在前面加一个空格，其余标点，在后面加一个空格（句末除外），标点之间不需要加空格。

4. 超链接两边加空格

当然，还是会有一些特殊情况的

例如：时间的书写，18:00, 2012/12/01

等等，需要自己灵活应变

标点符号

  • 半角标点符号
  • 全角标点符号
  • 前后都是中文句子时，用全角标点符号。
  • 前后都是英文句子时，用半角标点符号。
  • 前后分别为中文句子和英文句子，用全角和半角都可以。

错误：

当你在凝视 Bug 的时候,Bug 也在凝视你.

正确：

当你在凝视 Bug 的时候，Bug 也在凝视你。

错误：

He is a gay，you too！

正确：

He is a gay, you too!

正确：

储程（Eric, male）喜欢唱苦情歌。

正确：

储程 (Eric, male) 喜欢唱苦情歌。

工作中的错误案例

错误：

我们公司做过ERP、CRM、微信营销等。

正确：

我们公司做过 ERP, CRM, 微信营销等。

专有名词

  • 专有名词使用正确的大小写。
  • 不要使用自己创造的缩写。

正确：

使用 GitHub 登录

错误：

使用 github 登录

使用 Github 登录

使用 gitHub 登录

使用 GITHUB 登录

正确：

苹果更新了 iOS 系统

错误：

苹果更新了 ios 系统

苹果更新了 IOS 系统

苹果更新了 Ios 系统

苹果更新了 IoS 系统

正确：

我们需要招聘一个熟悉 JavaScript, HTML5, AngularJS 的前端开发者。

错误：

我们需要招聘一个熟悉 Js, h5, angular 的前端开发者。

段落

  • 文档不是作文，开头不要有留白
  • 段落之间应该留有空行

正确：

这是一个很长的段落。这是一个很长的段落。这是一个很长的段落。这是一个很长的段落。这是一个很长的段落。

这是一个很长的段落。这是一个很长的段落。这是一个很长的段落。这是一个很长的段落。这是一个很长的段落。

引用

如果在正文中部分引用第三方内容，请使用恰当的引用格式并给出出处。如：

One man’s constant is another man’s variable. — Alan Perlis

如果是全篇转载，请在全文开头显著位置注明出处并链接至原文，如：

本文转载自 WikiQuote

如果格式不允许超链接，请以文本方式直接给出原文链接。如果原文链接太长影响美观，可以使用短链接服务。如：

本文转载自 WikiQuote：http://bit.ly/UlHIdN

文案风格

  • 没有错别字。
  • 没有病句。
  • 我们崇尚精练的文风。请在检查中把对表达意思没有明显作用的字、词、句删除，在不影响表达效果的前提下把文案长度减到最短。

正确：

我赞成这个提议。

错误：

我十分且非常强烈得赞成这个提议！

错误：

哇，你说的太棒了，我迫不及待地要去赞成这个提议了！

以上书写规范同样适用于工作中的文字沟通
