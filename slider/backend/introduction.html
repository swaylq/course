理解 RESTful 架构

一句话概括，RESTful 描述的是在网络中前端和后端的一种交互形式，使用 URL 定位资源，用 HTTP 动词定义操作

数据库基础

是什么？什么作用？怎么用？

  • 数据库指的是以一定方式储存在一起，能为多个用户共享，具有尽可能小的冗余度，与应用程序彼此独立的数据集合。
  • 在 Web 开发中，数据库主要是为了持久性的存储网站的数据
  • 图形化管理界面; Commend Line ; ORM

数据库种类

  • 关系型数据库: MySQL
